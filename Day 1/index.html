DSA with JS - Day 1

Introduction

This document introduces you to the foundational concepts of JavaScript, focusing on basic operations, data types, and common JavaScript functions. These concepts are essential for learning Data Structures and Algorithms (DSA) in JavaScript.

1. Basics of JavaScript

Variables and Data Types

In JavaScript, variables store data values. Here's a look at the basic types of data:

  • Number: Numeric values. Example: let age = 30;
  • String: Sequence of characters. Example: let name = "John";
  • Boolean: true or false values. Example: let isStudent = true;
  • Object: Collection of key-value pairs. Example: let person = { name: "John", age: 30 };
  • Array: Ordered collection of values. Example: let fruits = ["apple", "banana", "cherry"];

2. Operators in JavaScript

Arithmetic Operators

let a = 10;
let b = 20;
console.log(a + b);  // 30
console.log(a * b);  // 200
console.log(b % a);  // 0 (remainder)

Relational Operators

let x = 10;
let y = 20;
console.log(x > y);    // false
console.log(x < y);    // true
console.log(x == 10);  // true

Logical Operators

let isAdult = true;
let hasPermission = false;
console.log(isAdult && hasPermission);  // false
console.log(isAdult || hasPermission);  // true

Unary Operators

let a = 5;
console.log(++a); // 6 (Pre-increment)
console.log(a--); // 6 (Post-decrement)

3. Type Coercion in JavaScript

Type coercion is JavaScript's ability to automatically convert values from one data type to another. For example:

let num = 5;
let str = "Hello";
console.log(num + str);  // "5Hello"
console.log("5" - 1);  // 4 (string "5" converted to number)

Type Casting

let num = "10";
let convertedNum = Number(num);
console.log(convertedNum);  // 10 (number)

Q3: Swap Two Variables Using 3 Methods

Method 1: Using a Temporary Variable

let num1 = 10;
let num2 = 20;
let temp = num1;
num1 = num2;
num2 = temp;
console.log(num1);  // 20
console.log(num2);  // 10

Method 2: Using Arithmetic Operations

let a = 10;
let b = 20;
a = a + b; // 30
b = a - b; // 10
a = a - b; // 20
console.log(a);  // 20
console.log(b);  // 10

Method 3: Using Destructuring Assignment

let x = 10;
let y = 20;
[x, y] = [y, x];
console.log(x);  // 20
console.log(y);  // 10

5. Math Functions in JavaScript

console.log(Math.round(4.7));    // 5
console.log(Math.ceil(4.3));     // 5
console.log(Math.floor(4.9));    // 4
console.log(Math.trunc(4.5));    // 4
console.log(Math.pow(2, 3));     // 8
console.log(Math.sqrt(25));      // 5
console.log(Math.abs(-10));      // 10
console.log(Math.max(1, 2, 3));  // 3
console.log(Math.min(1, 2, 3));  // 1
console.log(Math.random());      // Random number between 0 and 1
console.log(3.14159.toFixed(2)); // 3.14

6. Additional Practice

Q1: Calculate the Area and Perimeter of a Rectangle

let width = 10;
let height = 20;
let area = width * height;  // Area of rectangle
let perimeter = 2 * (width + height);  // Perimeter of rectangle
console.log("Area: " + area);         // Output: 200
console.log("Perimeter: " + perimeter);  // Output: 60

Q2: Generate a 4-Digit OTP

let otp = Math.trunc(Math.random() * 10000) + 1000;
console.log("OTP:", otp);  // Random 4-digit OTP

Q3: Calculate Area of a Triangle Using Heron's Formula

let sideA = 10;
let sideB = 20;
let sideC = 10;
let s = (sideA + sideB + sideC) / 2;  // Semi-perimeter
let area = Math.sqrt(s * (s - sideA) * (s - sideB) * (s - sideC));
console.log("Area of Triangle: ", area);

Q4: Circumference of a Circle

let radius = 30;
let circumference = 2 * Math.PI * radius;
console.log("Circumference: ", circumference);  // Output: 188.49555921538757
